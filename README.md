# 🌐 Aula 01 - Redes de Computadores | IFTO

Projeto desenvolvido para a prova didática da disciplina de Redes de Computadores no Instituto Federal do Tocantins (IFTO).

## 📚 Sobre o Projeto

Esta atividade tem como objetivo demonstrar, de forma prática e interativa, conceitos fundamentais relacionados às redes de computadores, utilizando uma aplicação web simples para representar:

- Compartilhamento de informações;
- Segurança de redes;
- Comunicação entre dispositivos;
- Protocolos de rede;
- Controle de acesso;
- Validação de informações;
- Uso de interfaces web como meio de interação em ambientes conectados.

O sistema utiliza uma palavra-chave específica para validar o acesso ao conteúdo, simulando mecanismos básicos de autenticação e segurança da informação.

---

## 🖥️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Vercel (Deploy)
